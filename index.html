Frequently Asked Questions

Pertanyaan-pertanyaan berikut menjelaskan ketentuan layanan kami agar Anda dapat menggunakan layanan kami dengan pemahaman yang jelas. Silakan baca FAQ ini sebelum Anda memutuskan untuk menggunakan layanan kami. Jika Anda memiliki pertanyaan tambahan, jangan ragu untuk menghubungi tim dukungan kami.

  • Apakah boleh shutdown Virtual Machine?

    Tidak, Anda tidak diperbolehkan untuk melakukan shutdown. Melanggar aturan ini akan mengakibatkan Virtual Machine Anda otomatis di-terminate dan tidak dapat mengajukan klaim garansi.

  • Apakah saya boleh menggunakan Virtual Machine untuk CHR MikroTik?

    Tidak, Anda tidak diperbolehkan menggunakan Virtual Machine untuk CHR MikroTik.

  • Apakah ada batasan penggunaan CPU?

    Ya, Anda dilarang menggunakan CPU hingga 100%. Penggunaan CPU yang berlebihan dapat memengaruhi kinerja Virtual Machine lainnya.

  • Dapatkah saya menggunakan VPN selain ekstensi Chrome?

    Tidak, Anda hanya diperbolehkan menggunakan VPN melalui ekstensi Chrome.

  • Apakah saya boleh melakukan aktivitas mining, hacking, atau torrenting?

    Tidak, Anda dilarang untuk melakukan aktivitas seperti mining, hacking, torrenting, atau aktivitas ilegal lainnya.

  • Apa yang dimaksud dengan "Bergaransi"?

    "Bergaransi" berarti bahwa kami menyediakan garansi penuh selama 30 hari. Ini berarti jika Anda mengalami masalah teknis selama periode ini, Anda dapat mengajukan klaim garansi.

  • Bagaimana cara klaim garansi?

    Anda dapat mengajukan klaim garansi jika Anda mengalami masalah teknis dengan layanan Virtual Machine. Garansi hanya dapat di-claim satu kali dan tidak akan memperpanjang masa aktif Virtual Machine Anda.

  • Apa yang akan terjadi jika saya menggunakan Virtual Machine untuk kegiatan ilegal?

    Garansi akan hangus jika Virtual Machine Anda digunakan untuk kegiatan ilegal. Kami tidak mendukung atau mengizinkan penggunaan Virtual Machine untuk aktivitas ilegal.
